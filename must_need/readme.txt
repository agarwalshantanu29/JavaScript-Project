Anthroprologie
Welcome to Anthroprologie, a women's clothing website. Our site features a clean and modern design, making it easy to navigate and find the perfect outfit.

Features
Landing page
Sign-in and sign-out functionality
This website is a clone created by Shantanu Agarwal.

Sign-in and Sign-out
To access all the features of the website, you will need to sign in.
If you are a new user, you can sign up by clicking on the "Sign up" button on the top right corner of the landing page.
Once you are signed in, you will be able to access the full range of products and make purchases.
To sign out, click on the "Sign out" button on the top right corner of the website.
Technical Details
The website is built using HTML, CSS, and JavaScript
The sign-in and sign-out functionality is implemented using Firebase Authentication.
